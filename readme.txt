A Machine Learning model which predicts the the price of a car depending on its condition,usage and features.
Simple linear regression, random forest regressor and decision tree regressor were implemented on the dataset out of which random forest proved to be accurate.
data set:- https://www.kaggle.com/nehalbirla/vehicle-dataset-from-cardekho
this is an end to end project complete with front end ui(vanilla html), backend (flask), model training and testing (google collab).
